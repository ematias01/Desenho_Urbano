## 📋 Sobre este Repositório

Este repositório reúne todos os arquivos necessários para a realização da **Atividade Prática** da disciplina de **Desenho Urbano**, do curso de **Arquitetura e Urbanismo** do Grupo SER Educacional.

Aqui você encontrará os arquivos de base para análise e desenvolvimento da atividade, incluindo a base cartográfica editável, o modelo de prancha oficial e as instruções completas.

## 🎯 Objetivo da Disciplina

A disciplina de **Desenho Urbano** explora os elementos que compõem a forma e a qualidade dos espaços urbanos — calçadas, praças, quadras, fachadas e gabaritos —, capacitando o estudante a analisar criticamente o ambiente construído e propor intervenções que melhorem a experiência das pessoas na cidade.

---

## 📌 Como Utilizar

1. Leia o arquivo **"Atividade prática — Desenho Urbano"** para entender o enunciado completo
2. Utilize a base `.dwg` (AutoCAD) ou `.rvt` (Revit) conforme o software disponível
3. Desenvolva sua proposta a partir da base cartográfica fornecida
4. Use o **modelo de prancha A2** (`.pptx`) para organizar e apresentar o trabalho final
5. Adapte o modelo com os dados do seu projeto antes da entrega
